## 🚀 使用流程

```
1. 安裝依賴
   → pip3 install -r requirements.txt

2. 配置 Telegram
   → 複製 .env.example 為 .env
   → 填寫 Bot Token 和 Chat ID

3. 測試系統
   → python3 tests/test_system.py

4. 啟動監控
   → python3 main.py

5. 接收信號
   → Telegram 即時推送進場通知
```

## 📈 預期效果

### 信號頻率
- **高質量信號：** 1-5 次/週
- **中等信號：** 5-10 次/週
- **低質量信號：** 被過濾（不推送）

### 信心度分佈
- **80-100%:** 極高質量（罕見）
- **70-79%:** 高質量（主要交易機會）
- **60-69%:** 中等質量（可選）
- **<60%:** 過濾

### 風險控制
- **單筆風險：** 1% 資本
- **總風險敞口：** ≤5%
- **最大回撤：** 15% 強制停止

## ⚠️ 重要提醒

### 使用前必讀
1. **先紙上交易** - 建議至少 30 天紙上交易驗證
2. **漸進式實盤** - 從 10% 頭寸開始，逐步增加
3. **風險控制** - 嚴格遵守 1% 單筆風險限制
4. **市場風險** - 極端行情下所有技術分析可能失效
